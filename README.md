# 🦇 Batman: El Caballero de la Noche

Proyecto web realizado con **HTML, CSS y JavaScript** basado en la película *Batman: El Caballero de la Noche*.

El objetivo del proyecto es practicar conceptos básicos de desarrollo frontend como:

- Estructura semántica en HTML
- Estilos y layouts con CSS
- Interactividad con JavaScript
- Manipulación del DOM
- Eventos y funciones
